好奇害死猫：12岁男孩尿道外口滴血20天

  • 添加收藏 返回上一页

  时间：2014-03-12

患者资料


       男性患儿，12岁，家长代诉尿道外口滴血20天，无外伤史，既往无类似病史。


初步检查结果


       入院后行泌尿系B超检查，无异常。


       查体：阴茎外观发育正常，触诊前尿道未及结石。


初步诊断


       前尿道血管瘤破裂出血？


尿道镜检查结果


       之后在静脉麻醉下行尿道镜检查，结果如图：


尿道内出现钉子样异物


       随后终止镜检，行X线检查：显示为针状异物。


       仔细询问病史：家长与患儿耐心沟通后，患儿诉20天前因好奇将一口缝被针塞入尿道外口，未能取出，也未告知家长。缝被针大约5-6厘米长。 




最终诊断 后尿道异物。


手术方案


       第二天，在骶管麻醉下经尿道异物取出术。


       第一套方案：以泌尿外科的异物钳去钳夹针尖。但因针的后半部分已扎入尿道海绵体内，无法固定针尖而失败。


       第二套方案：以消化内科的异物钳（带齿的）去钳夹针尖，仍无法固定针尖，失败。


       第三套方案：以普外科的胆道镜套石篮，将针尖套牢并固定，直视下缓慢后退，将针顺利成功取出。
